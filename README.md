# Proyecto – Preparación de Datos (NumPy + Pandas)

Pipeline completo de preparación de datos dividido en 6 lecciones:
1) Generación de datos ficticios (NumPy)  
2) Exploración inicial (Pandas)  
3) Integración de fuentes (CSV + Excel + HTML)  
4) Limpieza (nulos + outliers con IQR)  
5) Data Wrangling (duplicados, tipos, nuevas columnas, apply/map/lambda, discretización)  
6) Agregación y reestructuración (groupby, pivot_table, melt) + exportación final

## Estructura
- `notebooks/`: Notebook principal (.ipynb)
- `src/`: script ejecutable del pipeline (.py)
- `data/`: fuentes complementarias (Excel + HTML)
